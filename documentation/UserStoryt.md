## Käyttäjä

Käyttäjän pitää kirjautua sovellukseen, jonka jälkeen tämä voi tarkastella sovelluksessa olevia keskusteluita, lisätä niihin vastauksia sekä aloittaa uusia keskusteluja.

Käyttäjä pystyy poistamaan omia aloittamiaan keskusteluita ja lisäksi muokkaamaan kirjoittamiaan kommentteja.

Käyttäjä pystyy tilaamaan keskustelun, jolloinka se näkyy erillisellä tilaukset-sivulla. Tilauksen pystyy myös poistamaan.

## Admin

Admin pystyy suorittamaan samat tehtävät kuin käyttäjäkin ja tämän lisäksi admin pystyy poistamaan keskusteluja, vastauksia ja käyttäjiä kannasta.
